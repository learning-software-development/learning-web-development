## Using Eleventy

### Compile all files in current directory

```shell
eleventy
```

or

```shell
npx @11ty/eleventy
```

### Start up a web server

```shell
eleventy --serve
```

or

```shell
npx @11ty/eleventy --serve [--port=8081]
```

### Command Line Help

```shell
npx @11ty/eleventy --help
```
